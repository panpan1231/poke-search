# Pokex — 寶可夢屬性相剋查詢

純前端單頁工具，快速查詢寶可夢屬性弱點、建立隊伍並取得對戰建議。

## 功能

### 屬性查詢

選擇 1～2 個屬性，立即顯示：

- **弱點** — 被哪些屬性剋制（×2 / ×4）
- **抵抗** — 對哪些屬性減傷（×½ / ×¼）
- **免疫** — 完全無效的屬性（×0）

### 我的隊伍

建立最多 **3 組隊伍**，每隊最多 **4 隻寶可夢**：

1. 切換到「我的隊伍」分頁
2. 輸入寶可夢名稱（選填，方便辨識）
3. 點選該寶可夢的 1～2 個屬性
4. 按「加入隊伍」

隊伍資料自動儲存在瀏覽器 localStorage，關閉頁面也不會消失。

### 隊伍分析

按下「分析隊伍弱點與攻略」後顯示：

- **各成員弱點一覽** — 每隻寶可夢的弱點屬性及倍率
- **對戰情境分析** — 列出威脅多隻隊員的危險屬性，並推薦反擊招式
- **推薦攜帶的招式屬性** — 依威脅權重排序，確保招式覆蓋率

### 即時對手分析

隊伍分析完成後，下方會出現「即時對手分析」：

1. 選擇當前對手的 1～2 個屬性
2. 即時顯示每隻隊員的對戰評估：
   - 🟢 **最佳人選** — 能打超效且不怕對手
   - 🟢 **能擋住** — 抵抗對手攻擊
   - 🟡 **可打但危險** — 能剋制對手但自己也會被打
   - 🔴 **避免上場** — 會被對手剋制
   - ⚪ **免疫對手** — 完全不受傷害
3. 底部列出所有能剋制對手的招式屬性

## 安裝為 App（PWA）

本工具支援 PWA，可以安裝到手機桌面像原生 App 一樣使用，離線也能運作。

**手機（iOS Safari）：**

1. 用 Safari 開啟頁面
2. 點選分享按鈕 → 「加入主畫面」

**手機（Android Chrome）：**

1. 用 Chrome 開啟頁面
2. 點選選單 → 「安裝應用程式」

**桌面（Chrome / Edge）：**

- 網址列右側會出現安裝圖示，點擊即可安裝

> 注意：PWA 需要透過 HTTP 伺服器存取（`localhost` 或 `https`），直接開啟 `file://` 無法啟用 Service Worker。本機測試可用：
>
> ```
> npx serve .
> ```

## 使用方式

直接在瀏覽器開啟 `index.html` 即可使用基本功能。若需要 PWA 離線功能，請透過本機伺服器存取。

## 技術細節

- 單一 HTML 檔案（HTML + CSS + JS 全包）
- 18 種屬性的攻防倍率表完整內建
- 繁體中文介面
- Mobile-first RWD 設計，手機與桌面皆適用
- 資料持久化使用 localStorage
